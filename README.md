## 🚀 Projects

### ⚡ Calton — 3-Phase AC Energy Monitor *(in progress)*
Custom hardware energy monitoring system using the **ADE9000** metering IC and **STM32H743** microcontroller. Full custom PCB with isolated amplifiers (AMC1301), MOV surge protection, and SPI4 communication. Targeting sub-1% measurement accuracy across voltage and current channels.
> `STM32` `ADE9000` `SPI` `PCB Design` `KiCad` `Power Metering`

---

### 🔍 [Li-Sense-Detect](https://github.com/m4g3shw4r/Li-Sense-Detect) — Real-Time ANPR
Automatic Number Plate Recognition prototype using **PaddleOCR + OpenCV**, targeting Raspberry Pi deployment. Live camera capture, Indian plate format validation, CSV/SQLite event logging, and a CLI query tool.
> `Python` `PaddleOCR` `OpenCV` `SQLite` `Edge ML`

---

### 🌡️ [Energy-Automation-PID-Building-Simulation](https://github.com/m4g3shw4r/Energy-Automation-PID-Building-Simulation)
Occupancy-based lighting + PID HVAC temperature control simulated over 24 hours in MATLAB/Octave, with full energy (kWh) and cost reporting. Built to understand real building automation control loops.
> `MATLAB` `PID Control` `Energy Systems` `Simulation`

---

### 📶 [wifi-beacon-scanner](https://github.com/m4g3shw4r/wifi-beacon-scanner)
Low-level 802.11 beacon frame scanner in C. Captures and parses raw Wi-Fi management frames using raw sockets.
> `C` `Linux` `Raw Sockets` `802.11` `Networking`

### 🔁 [ring-buffer-uart-sim](https://github.com/m4g3shw4r/ring-buffer-uart-sim)
Circular buffer implementation for UART communication, simulated in C. Classic embedded data structure for interrupt-driven serial I/O without memory allocation.
> `C` `Embedded` `UART` `Data Structures`
